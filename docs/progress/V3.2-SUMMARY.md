# V3.2 — New-player onboarding (phase boundary)

**Commit**: `5c8e0e9`
**Status**: ✅ complete, 391 tests passing, prod build green.

---

## Changes

1. **Catalog reorder** — Sklepik moved before Mała elektrownia in
   `BUILDING_CATALOG`. Sklepik is the easier first build (50 coins
   earned lifetime = any game; yields coins + bricks so it bootstraps
   the econ). Mała elektrownia stays step 2.

2. **Unlock shortcut** — Mała elektrownia now unlocks via either
   earning 50 watts OR owning Sklepik. New `UnlockCondition` kinds:
   `has-building` + `any-of` (recursive). `isUnlocked` signature gained
   an optional `buildings` parameter (back-compat default `[]`).

3. **Starter kit** — `ensureSignupGift` credits `{ coins: 50, bricks: 50 }`
   on first call. Idempotent via ledger SADD on sourceId
   `signup:starter-kit`. Feature flag `v3_starter_kit` (default on).
   New helper `hasReceivedStarterKit(username)`.

4. **Register route** — fires `ensureSignupGift` synchronously after
   session creation so first render already has Domek + wallet.

## Feature flag

`v3_starter_kit` default on. Flipping off leaves Domek as the only
signup gift (no wallet credit). Idempotency holds either way.
