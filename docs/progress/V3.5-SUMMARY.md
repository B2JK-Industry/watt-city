## Changes

- `Loan.autoRepay?: boolean` (optional, default-true for new loans, undefined legacy = true).
- `processLoanPayments` — skips auto-deduct path when `autoRepay === false`; still counts miss + credit-score penalty.
- `PATCH /api/loans/{id}/auto-repay` — zod body `{enabled}`, session + loan-active guards.
- `components/loan-schedule.tsx` — dashboard widget. Shows next payment date + amount + days + progress bar + status + toggle. 4-lang copy.
- Dashboard mounts `LoanSchedule` when `player.loans.length > 0`.
- Tests (+4): autoRepay default, auto-deduct respected, autoRepay=false forces miss even with funds, legacy-record fallback.
