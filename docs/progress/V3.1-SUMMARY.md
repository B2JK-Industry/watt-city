## What V3.1 did

Replaced the V1/V2 XP-Arena tier narrative (a single growing avatar
building, `CITY_TIERS` nine-tier gallery, `tierForLevel` emoji badges)
with a pure city-first surface — the player's real 20-slot city is
now the identity everywhere it appears (dashboard hero, nav badge,
/o-platforme ladder, /games header, landing).

## Feature flag

`v3_city_first` default **on**. Flipping off disables the city-first
badge in the nav (falls back to "XP Lv N" placeholder) and the
CityLevelCard mount on dashboard. CITY_TIERS types are gone, so the
fallback can't restore the original tier UI — this is deliberate per
design doc ("V3 is the one we want in prod; flag is a kill-switch
not a dual-UI toggle").
